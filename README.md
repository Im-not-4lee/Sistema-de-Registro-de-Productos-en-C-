# Sistema de Registro de Productos en C++

Proyecto académico que implementa el manejo de archivos binarios
usando estructuras en C++.

## Funcionalidades
- Registro de productos
- Lectura de productos almacenados
- Conteo de registros en archivo binario

## Tecnologías
- C++
- Archivos binarios (`fstream`)
- Estructuras (`struct`)

## Compilación
g++ main.cpp -o registro
